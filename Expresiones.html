Expresiones

Las expresiones en Angularjs están contenidas dentro de {{ }}.

Las expresiones enlazan los datos a los elementos HTML. Únicamente interpretado dentro de la directiva ng-app

ng-app instalado

El article de abajo tiene instalada la directiva ng-app

La suma es {{ 5+5 }}

Inicializando valores con ng-init

Con ng-init se inicializan valores de las directivas

El div de abajo tiene inicializado el color de la letra con clases de materialize

La clase para el color de este párrafo es {{ myColor }}
