Las directivas amplian el uso de las etiquetas y atributos HTML. Las directivas básicas son: ng-app, ng-model, ng-bind,ng-init

ng-app: Define una aplicación AngularJs en el elemento html donde se declare y sus hijos heredan este comportamiento.

ng-model: Permite vincular los valores de los elementos HTML a variables de angular para validarlos, ver su status y enlazarlos a la capa de datos.

ng-bind:Vincula el valor de los elementos HTML a variables de Angular.

ng-init:Inicializa los valores de la aplicación.
