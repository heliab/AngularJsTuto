Añadiendo AngularJs

Para usar AngularJs en el documento HTML, se puede hacer de las siguientes maneras:

Desde repositorio online 
https://ajax.googleapis.com/ajax/libs/angularjs/1.5.2/angular.min.js
Desde bower 
bower install angular#1.5.2
Desde npm 
npm install angular@1.5.2

Puede descargar AngularJs desde: AngularJs

Las expresiones enlazan los datos a los elementos HTML. Solo se reconocen cuando se encuentra declarado en el elemento padre la directiva ng-app. Este puede ser la etiqueta html principal para que todo el documento sea interpretado como una aplicación Angularjs. 

									
<html ng-app>
  <head>
    <title></title>
  </head>
  <body>
  </body>
</html>
